Description

1. Defect inspection must be carried out multiple time starting from 3 weeks before handover and the last one must be the week before handover The following team members must attend the defect inspection PM, SM, Operation Manager and Head Designer and Head of Construction for the last 2 2. Defects must be managed through Procore via the defect list tool a) Each defect item should have a link to a drawing (one drawing must be picked by the team, ideal layout and defect should start with a pin on the drawing) b) Each defect item should have a description of the defect c) Each defect item should nominate the which trade this defect has been addressed to. d) Each defect item should have photos attached to it. e) Each defect item should have location f) Each defect item should be assigned to a contractor g) Defect must be closed once contractor completed the defect and provided evidence of completion.
