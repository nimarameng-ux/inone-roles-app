Below is the current structure of the Project Closure Folder and what needs to be saved in each folder as a minimum. 1. Head Contract Financials This folder must include the Financial Report in both PDF and Excel. The financial report must cover: a) Tender margin b) Letting gain margin c) Final margin d) Revised contract value e) List of approved variations f) List of rejected variations g) List of internal variations h) Confirmation that the entire contract value has been claimed i) Confirmation that all approved variations have been claimed (and noted if any are pending for a specific reason) 2. Supp Units) m) Training Electrical n) Training AV (provide quick reference guides to be printed, laminated and left on the boardroom table) o) Training Security p) Client issued PC Certificate 4. Architectural drawings 15. Practical completion certificate 16. AV 17. Furniture Then: c) Save a PDF copy of the Procore correspondence in this folder. d) Copy the ZIP handover pack itself into this folder as well. 5. CDC & OC a) Save copies of CDC, CC, Building Permit and OC (or CFI) in this folder.
