# INONE Projects — Roles & Responsibilities App

## How to use
1. Open the app link (Vercel URL).
2. Use the Role Dashboard to filter tasks by role (PM / SM / OM / Tendering / Design etc.).
3. Search by keyword using the search bar.
4. Click any task to open its dedicated page (description + responsibility table).
5. Optional: Click “Copy link” to share a specific task page.
6. Optional: Install as an app:
   - Chrome / Edge → click the Install icon in the address bar.

## Tips
- If the app looks outdated after an update: hard refresh (Cmd+Shift+R).
